# Hideout Cat
Created for [S.T.A.L.K.E.R. Anomaly](https://www.moddb.com/mods/stalker-anomaly/) and [Hideout Furniture](https://www.moddb.com/mods/stalker-anomaly/addons/hideout-furniture-hf).

This mod adds: 

- Alarm System — placeable alarm that scan for nearby enemies/monsters.
- Pegboard — wall display with 3 weapon slots for rifles/snipers/shotguns/smgs, usable as a weapon display/storage spot.
- Pistol Pegboard — variant of the pegboard with 9 pistol slots.
- Headgear Display — display slot for a helmet.
<img width="1354" height="591" alt="Displays" src="https://github.com/user-attachments/assets/d3467191-255d-4b2f-8a16-737d31b343f6" />
- Artifact Harvester — device that creates resources over time from inserted artifacts.
<img width="1119" height="593" alt="Harvester" src="https://github.com/user-attachments/assets/7e4e17f9-4f04-4309-8c0b-6c48c63f3c04" />
- Trading Computer — laptop that opens a trader-selection UI and starts trade with a selected local trader.
